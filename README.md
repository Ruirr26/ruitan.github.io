# Rui Tan Personal Website

This is a static academic personal website prepared for GitHub Pages.

这是一个可以直接用于 GitHub Pages 的个人学术主页项目。

## Project Files

- `index.html` - homepage content
- `styles.css` - visual styling
- `script.js` - mobile navigation
- `assets/rui-tan-headshot.jpg` - profile photo
- `assets/rui-tan-cv.pdf` - CV
- `assets/favicon.svg` - browser icon
- `.nojekyll` - keeps GitHub Pages from changing asset paths

## Publish Option 1: User Site

Use this option if you want the website address to look like:

`https://your-username.github.io/`

1. Create a GitHub repository named exactly `your-username.github.io`.
2. Upload every file in this folder to the repository root.
3. Open `Settings > Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Save.

## Publish Option 2: Project Site

Use this option if you want to keep any repository name, for example `personal-website`.

The website address will look like:

`https://your-username.github.io/personal-website/`

1. Create a GitHub repository, for example `personal-website`.
2. Upload every file in this folder to the repository root.
3. Open `Settings > Pages`.
4. Choose branch `main` and folder `/ (root)`.
5. Save.

## 中文发布步骤

最简单的方法：

1. 在 GitHub 新建仓库。
2. 如果想要网址是 `https://你的用户名.github.io/`，仓库名必须是 `你的用户名.github.io`。
3. 把本文件夹里的所有文件上传到仓库根目录。
4. 打开仓库的 `Settings > Pages`。
5. 选择 `Deploy from a branch`，分支选 `main`，目录选 `/ (root)`。
6. 等 1-3 分钟，GitHub 会生成网站链接。
